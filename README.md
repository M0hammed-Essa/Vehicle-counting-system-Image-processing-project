# Vehicle counting system-Image processing- project

## Idea description

This project presents an automated vehicle counting system using image processing and computer vision techniques. The system analyzes video frames or image sequences from a traffic scene to detect and count moving vehicles such as cars, trucks, and buses.

It applies background subtraction to separate moving objects from the static environment, followed by noise reduction and morphological operations to improve detection accuracy. Once vehicles are identified, the system tracks and counts them over time.

The output provides the total number of vehicles, which can be used for traffic monitoring, congestion analysis, and smart city applications.
